# Student Management API

This is a Spring Boot-based REST API for managing student records.

## Features
- Add student
- Update student
- Delete student
- Get student details
- List all students

## Technologies Used
- Spring Boot
- Spring Data JPA
- Postgresql
- REST API

### Prerequisites
- Java 17+
- Maven
